在项目中添加I18nLib

首先，我们在 Visual Studio 中命名并创建一个.net framework 项目 或.net core 项目，在这里我们将其命名为 Example，右击引用 选择添加引用-浏览，单击下面的浏览，在弹出的窗 口中选择我们刚刚生成的 i18nLib.dll 库。即可在您的项目中添加并 使用 I18nLib 库。
